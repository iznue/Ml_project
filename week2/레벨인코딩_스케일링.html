▶ [문자-숫자변환] 레벨인코딩 (교재 117)¶

  • 문자로된 y값을 숫자로 변형해야할때 필수
    • fit은 값은 유니크하게 바뀌면서 0,1,2,3..값을 할당
    • transform은 fit된 값을 y값에 적용
  • 트레이닝하고 테스트나누기전에 처음부터 숫자로 변경한뒤, 트레인과 테스트 자료를 나누는걸 권장
  • 만약 나뉘어져 있는 자료라면 트레이닝에만 fit, transform을 다 하고,
    • 테스트는 transform만 진행

items=('tv, 냉장고, 전자레인지, 컴퓨터, 선풍기, 선풍기, 믹서, 믹서').split(',')
from sklearn.preprocessing import LabelEncoder
encoder=LabelEncoder()
encoder.fit(items)    ###### items를 중복제거하고 0,1,2,3,4의 순서번호를 갖게함.
print(encoder.classes_)
encoder.transform(items)  ### fit해서 나온 결과물을 items에 적용
Out[27]:
array([5, 0, 3, 4, 2, 2, 1, 1])

  • 카테코리에 해당하는 문자변수를 낱개의 필드(x값)로 쪼갬
